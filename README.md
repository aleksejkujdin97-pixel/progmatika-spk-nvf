# Корпоративный сайт ООО «ПРОГМАТИКА»

В этом репозитории находится полностью реализованный код корпоративного сайта для компании, специализирующейся на
фасадных системах — навесных вентилируемых фасадах (НВФ) и светопрозрачных конструкциях (СПК). Сайт разработан на
**Next.js** с использованием App Router и React Server Components и соответствует высокому уровню
Enterprise‑архитектуры, заданному в ТЗ.

## Особенности реализации

- **Глубокая информационная архитектура:** все разделы из ТЗ реализованы: «Платформа», «Отрасли», «Кейсы»,
  «Технологии», «Аналитика и документы», «О компании», «Карьера» и «Контакты». Навигация интуитивна, а контент
  разбит на карточки и блоки, упрощающие восприятие.
- **Модульность:** компоненты (`Hero`, `CaseCard`, `TechCard`, `JobCard`, `IndustryCard`, `DocumentCard`,
  `FilterBar`) написаны как переиспользуемые модули и подключаются в соответствующих страницах.
- **Дизайн‑система:** дизайн‑токены (цвета, отступы, радиусы) описаны в `globals.css`, с поддержкой светлой и
  тёмной тем. Вёрстка адаптивна и масштабируется.
- **Данные:** примеры кейсов, технологий, отраслей, вакансий и документов размещены в `src/data`. В реальном
  проекте они могут быть заменены на подключение к headless CMS или базе данных.
- **Формы:** реализована форма пресейла/обратной связи, которая отправляет данные на API‑маршрут `/api/contact`.
- **Контактная информация:** раздел «Контакты» содержит адрес, телефон, email и форму запроса.
- **Вакансии:** страница «Карьера» выводит актуальные вакансии с описанием обязанностей и требований.

## Структура проекта

- **app/** — страницы и API‑маршруты. Каждая папка внутри `app` соответствует маршруту; файлы `layout.tsx` и
  `globals.css` задают общие компоненты и стили.
- **src/components/** — библиотека UI‑компонентов.
- **src/data/** — статические данные для кейсов, технологий, отраслей, вакансий и документов.
- **src/lib/** — вспомогательные типы и перечисления.
- **middleware.ts** — место для подключения дополнительных механизмов (аудит, rate‑limiting).

## Как запустить

1. Установите Node.js версии ≥ 18.
2. Установите зависимости `npm install` (может потребоваться доступ к npm‑registry).
3. Запустите проект командой `npm run dev`.
4. Откройте `http://localhost:3000` в браузере.

## Потенциал развития

- **Интеграция с CMS:** подключение headless CMS (например, Strapi) для хранения контента и вакансий.
- **Расширение CRM:** доработать API‑маршруты для отправки данных в CRM и ERP.
- **Мультиязычность:** добавить поддержку нескольких языков.
- **Оптимизация SEO:** расширить метаданные, включить генерацию sitemap и robots.txt.
- **Тестирование и CI/CD:** внедрить юнит‑ и E2E‑тесты, настроить автоматический деплой.

## Источники

Некоторые описания технологий в проекте основаны на открытых источниках: 
- Современные системы НВФ состоят из облицовочных панелей, закреплённых на металлическом каркасе, с
  вентиляционным зазором, удаляющим влагу【946571961841806†L197-L204】.
- Светопрозрачные фасады (СПК) используют профили из алюминия или стали и стеклопакеты, обеспечивая
  естественное освещение зданий【63643936511437†L13-L24】.
